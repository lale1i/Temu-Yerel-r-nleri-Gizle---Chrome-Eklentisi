# Temu Yerel Ürünleri Gizle---Chrome-Eklentisi
temu.com'da yerel mağazaları(türkiye mağazaları) listenin gerisine atarak yabancı(çin vs.) menşei ürünleri listenin başında gösteren tarayıcı eklentisi. Bu sayede yabancı menşei ucuz fiyatlı ürünlere kolayca erişebilirsiniz.
<br><br>
Kullanmak için dosyayı indirip zip'ten çıkartın. Daha sonra chrome'da eklenti ekleme kısmından ekleyin.
<br><br>
Anasayfada sayfayı yenileyerek veya arama kısmında herhangi bir ürün arayarak Yabancı Menşei ürünlere erişebilirsiniz.
